# Score Viewer Frontend

## Run locally for develop
- `npm install`
- `npm start`
- `http://localhost:3004`

### Requirements
- _node_: `16.15.0`
- _npm_: `8.5.5`

## Run via Docker
- `docker build -t source-view-frontend .`
- `docker run -p 3003:3000 source-view-frontend`
- `http://localhost:3003`
